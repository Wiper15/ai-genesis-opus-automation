# Opus Platform Setup Guide

## 1. Account Creation
- Visit: https://www.opus.ai
- Sign up for free account
- Verify email address

## 2. Platform Orientation
- Explore the visual canvas interface
- Try dragging different node types:
  - Data connectors
  - AI agents  
  - Decision nodes
  - Review steps
  - Export nodes

## 3. First Practice Workflow
1. Create new workflow
2. Add "Data Input" node
3. Connect to "AI Agent" node
4. Add "Human Review" node
5. Connect to "Export" node
6. Run test workflow

## 4. Key Resources
- Opus documentation: https://docs.opus.ai
- Tutorial videos: Available in platform
- Community support: Opus Discord channel
